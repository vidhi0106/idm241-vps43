Microinteraction: Reactions on LinkedIn

TRIGGERS

Hover on Like Button

Hover off Like Button

Hover on Reaction Buttons

Hover off Reaction Buttons

Click on Like Button

Click on Reaction Buttons

RULES

Hover on Like Button
  • The reaction modal pops up.
  • The reaction buttons pop up and then come down in a wave format one time.
  • Grey box appears behind the Like button.
  • Cursor changes to pointer.

Hover off Like Button
  • Pointer changes back to cursor.
  • There is a small delay and the reaction modal disappears.

Hover on Reaction Buttons
  • The reaction button you hover on slides up and expands 1.5 times its original size.
  • A text modal describing the reaction button fades in on top of the reaction button.
  • All other reaction buttons shrink by 0.75 times their original size.

Hover off Reaction Buttons
  • There is a small delay and the reaction modal disappears along with the text modal.
  • Pointer changes back to cursor.

Click on Like button
  • The Like Icon has a blue fill.
  • The text 'Like' turns blue.
  • The number of likes is accounted by logging the name of the user that hit the like button - database is updated.

Click on Reaction Buttons
  • The Like Icon is replaced by the icon of the reaction button.
  • The text 'Like' replaced by the text describing the reaction button.
  • The text color is the same as the color of the reaction button
  • The number of reactions is accounted by logging the name of the user that hit the like button - database is updated.
  • The type of reaction is accounted by logging the name of the user that hit the like button - database is updated.
