RULES

Hover on Like Button

- The reaction modal pops up
- The reactions pop up and then come down in a wave format one time
- Grey box appears on the Like button
- Cursor changes to pointer

Hover off Like Button

- Pointer changes back to cursor
- A small delay and the reactions disappear
